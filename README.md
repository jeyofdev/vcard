## Prerequisites

- Angular 19
- Typescript 5.7 or higher

## Installation

1. Clone the repository :
    ```sh
    git clone https://github.com/jeyofdev/vcard.git
    cd vcard
    ```

2. Run the application:
    ```sh
    bun run start
    npm run start
    ```
